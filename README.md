# Instalador de APK

Aplicación Android para gestionar la instalación de archivos APK.

## Características

- Instalación de APKs
- Gestión automática de permisos
- Interfaz de usuario intuitiva
- Soporte para Android 6.0 y superior

## Requisitos

- Android Studio Arctic Fox o superior
- Android SDK 24+
- Gradle 7.0+

## Instalación

1. Clonar el repositorio
2. Abrir en Android Studio
3. Sincronizar proyecto con Gradle
4. Ejecutar en dispositivo o emulador
